# Titanic Dataset Exploratory Data Analysis

## Objective
Analyze the Titanic dataset to understand factors affecting passenger survival.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data Exploration
- Missing Value Analysis
- Survival Count
- Survival by Gender
- Age Distribution
- Passenger Class Analysis
- Correlation Heatmap

## Key Insights
- Female passengers survived more than males.
- First-class passengers had a higher survival rate.
- Age and passenger class influenced survival.
- The dataset contains missing values in Age, Cabin, and Embarked columns.
