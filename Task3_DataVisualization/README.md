# CodeAlpha - Data Visualization

## Project Overview

This project is completed as part of the CodeAlpha Data Analytics Internship.

Dataset Used:
- Titanic Dataset

Libraries Used:
- Python
- Pandas
- Matplotlib
- Seaborn

Visualizations Created:
1. Survival Count (Bar Chart)
2. Gender Distribution (Pie Chart)
3. Age Distribution (Histogram)
4. Passenger Class Distribution (Count Plot)
5. Correlation Heatmap

Insights:
- More passengers died than survived.
- Most passengers were males.
- Majority of passengers belonged to third class.
- Most passengers were between 20 and 40 years old.
- Survival is related to passenger class and fare.
