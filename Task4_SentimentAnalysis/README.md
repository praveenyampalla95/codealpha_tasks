\# CodeAlpha Task 4 - Sentiment Analysis



\## Project Overview



This project is completed as part of the CodeAlpha Data Analytics Internship.



\### Dataset

Amazon Sentiment Spectrum Reviews Dataset



\### Tools Used

\- Python

\- Pandas

\- Scikit-Learn



\### Algorithms

\- CountVectorizer

\- Multinomial Naive Bayes



\### Features

\- Text Vectorization

\- Sentiment Classification

\- Accuracy Evaluation

\- Classification Report

\- Confusion Matrix



\### Results

\- Accuracy: 85.13%

\- Classified reviews as Positive or Negative.
